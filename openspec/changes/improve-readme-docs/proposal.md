## Why

README.md на GitHub — первое, что видит пользователь. Сейчас описание слишком краткое, а секция подключения к IDE содержит только 3 сервера из 9, что может ввести в заблуждение. Нужно: развёрнутое описание проекта, полный пример конфигурации MCP для IDE, и ссылка на официальную документацию.

## What Changes

- Расширить описание проекта в README.md — что такое MCP-серверы, зачем нужны, как помогают
- Переписать секцию «Подключение к IDE» — полный конфигурационный файл со всеми серверами, пояснения для разных IDE (Cursor, Claude Code, Windsurf), описание GitHub-репозитория на странице
- Добавить описание репозитория (About) на GitHub
