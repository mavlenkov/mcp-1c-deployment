## ADDED Requirements

### Requirement: Таблица-легенда серверов в разделе IDE
Раздел "Подключение к IDE" SHALL содержать таблицу с кратким описанием каждого MCP-сервера: имя в конфигурации, порт, назначение (1 строка), и пометку "рекомендуется" для ключевых серверов.

#### Scenario: Пользователь видит описание серверов перед JSON
- **WHEN** пользователь читает раздел "Подключение к IDE"
- **THEN** перед JSON-примером он видит таблицу с описанием каждого сервера и его назначением

### Requirement: Минимальная конфигурация
Раздел SHALL содержать пример "Минимальная конфигурация" с 3-4 основными серверами (help, graph-metadata, metadata, syntax) и пояснением, что остальные можно добавить по необходимости.

#### Scenario: Пользователь начинает с минимального набора
- **WHEN** пользователь хочет быстро начать работу
- **THEN** он копирует минимальный пример и получает работающую конфигурацию с основными возможностями

### Requirement: Полная конфигурация с пояснениями
Раздел SHALL содержать полный пример конфигурации со всеми серверами, с текстовым пояснением перед ним о том, что это расширенный вариант.

#### Scenario: Пользователь подключает все серверы
- **WHEN** пользователь хочет полный набор возможностей
- **THEN** он находит полный пример конфигурации со всеми 8 серверами
