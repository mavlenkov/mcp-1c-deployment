## Why

Раздел "Подключение к IDE" в README содержит JSON-пример конфигурации со всеми 8 серверами, но без пояснений — непонятно, какой сервер за что отвечает и какие из них обязательны. Также нужно вынести ссылку на официальную документацию в более заметное место.

## What Changes

- Добавить комментарии/описания к каждому серверу в примере конфигурации IDE, чтобы пользователь понимал назначение каждого сервера
- Разделить пример на "минимальную" и "полную" конфигурацию для разных сценариев использования
- Вынести ссылку на официальную документацию (https://docs.onerpa.ru/mcp-servery-1c) в шапку README рядом с другими важными ссылками
- Добавить краткие пояснения о приоритетности серверов (какие подключать в первую очередь)

## Capabilities

### New Capabilities
- `ide-config-docs`: Улучшенная документация по подключению MCP-серверов к IDE с описанием каждого сервера и рекомендациями по выбору

### Modified Capabilities
- `readme-improvements`: Дополнение раздела подключения к IDE и размещение ссылки на документацию

## Impact

- Изменения только в `README.md` — документация, без изменений кода или конфигурации
- Улучшает onboarding новых пользователей
